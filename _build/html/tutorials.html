Tutorial¶

The following tutorials walk you through performing FDTD simulation using the Dupid library.

There are 4 basic objects in a Dupid Simulation.

  • Simulation the simulation object

  • DupidSource the source object which creates the electromagnetic waves in the simulation

  • DupidObject the materials with which the electromagnetic waves will interact

  • detector the detector which takes reading during the running of the simulation
